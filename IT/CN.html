About this course

In this subject Computer networks we are going to learn on how two are connected. As we know devices can be either connected using a wired medium or wireless medium for the purpose of sending and receiving of data with others.
There are multiple devices helps in communication between two devices which are known as Network Devices. Ex:- Switch, Router, Hub.

The Pattern which device are interconnected are called Network Toplogy. Some Mostly used Topology are :- Star, Bus, Ring, Mesh.
We will also be studying the basic architecture of Network models. In that we studying how data transferred among two different devices. what models are used to connect two devices. Some of the models used to connect devices are as given below :

1.OSI MODEL:-
OSI (Open Systems Interconnection) model was created by the International Organization for Standardization (ISO), It describes how network applications on different computers can communicate through the network media. It consist of 7 layers:- Physical Layer, Data Link Layer, Network Layer, Transport Layer, Session Layer, Presentation Layer, Application Layer.

2.TCP/IP MODEL:-
The TCP/IP model was created in the 1970s by the Defense Advance Research Project Agency (DARPA) . it describes general guidelines for designing and implementing computer protocols. It consists of four layers: Network Access, Internet, Transport, and Application.

Protocol: A protocol is the set of rules or algorithms which define how two devices can communicate across the network and there exists different protocol defined at each layer of the OSI model. Few of such protocols are TCP, IP, UDP, ARP, DHCP, FTP and so on.

Media Access Control (MAC) address is a 48-bit (6 bytes) address that is used for communication between two hosts in an Network environment. It is a hardware address, which means that it is stored in the firmware of the network card.

IP(INTERNET PROTOCOL) address is a 32-bit number that identifies a host on a network. Each device that wants to communicate with other devices on a TCP/IP network needs to have an IP address configured. For example, if you want to access the Internet, your computer will need to have an IP address assigned

Implementation of the subject :

Computer networks is implemented in our Day-to-Day life. To understand the connectivity between any two devices, you need to know about the “Computer Networks”.

There are different Network types based on the geographical areas they occupy and also based on the number of devices used in that Network. Some of the different networks based on size are:

• Personal area network, or PAN – We use in Houses.
• Local area network, or LAN – We use in Offices
• Metropolitan area network, or MAN – We use in metros or Malls.
• Wide area network, or WAN – We use to connect a whole City.

Job trends:

If we choose to work in the domain of Computer Networks we can join in the following positions.

•Network Administrator - A network administrator is responsible for keeping an organization's computer network up-to-date
•Network Analyst - A network analyst will analyse and configure a company or organization's network, hardware, and computer systems and make any necessary changes or repairs
•Network Manager - A network Manager is responsible for managing a network.
