Description
About this course

An Algorithm is a sequence of steps to solve a problem. Design and Analysis of Algorithm is very important for designing algorithm to solve different types of problems in the branch of computer science and information technology. An algorithm is an efficient method that can be expressed within finite amount of time and space. An algorithm is the best way to represent the solution of a particular problem in a very simple and efficient way. If we have an algorithm for a specific problem, then we can implement it in any programming language, meaning that the algorithm is independent from any programming languages. So Designing and analyzing the algorithm plays an important role. The unit wise description of the subject is as given below:-

UNIT 1: INTRODUCTION
This unit deals with the Notion of an algorithm, Fundamentals of Algorithmic Problem Solving, Fundamentals of the Analysis of Algorithmic efficiency and Analysis Framework. Most important part of this unit is Asymptotic Notations and its properties and performing Mathematical analysis of Recursive and Non-Recursive Algorithms
UNIT 2:BRUTE FORCE AND DIVIDE-AND-CONQUER
By this unit we come across the Problem-Solving Techniques. Under Brute Force we come across problems like Closest-Pair and Convex-hull, Travelling Salesman problem, Knapsack problem and Assignment problem. Under Divide and Conquer we come across problems like Merge Sort, Quick Sort, Binary Search, Multiplication of Large Integers and Strassenâ€™s Matrix Multiplication
UNIT 3:DYNAMIC PROGRAMMING AND GREEDY TECHNIQUE
This unit deals how Dynamic Programming solves problems by combining the solutions of sub problems and Greedy Technique follows local optimal choice at each step with the hope of finding global optimum solution Under Dynamic Programming we come across problems like Computing Binomial Coefficient, Knapsack problem and Memory functions Under Greedy Technique we come across problems like Prim,Kruskal,Dijksta;s Algorithms and Huffman Trees
UNIT 4:ITERATIVE IMPROVEMENT
This unit deals how Iterative Improvement approaches a solution by progressive approximation and how it finds the (k+1)th approximate solution using kth approximate solution. We come across problems like Simplex Method, Maximum Flow problem, Maximum Matching in Bipartite Graphs and Stable Marriage Problem
UNIT 5:COPING WITH THE LIMITATIONS OF ALGORITHM POWER
This unit deals how to identify the Algorithms Limits by using Lower-Bound Arguments, Decision Trees, P,NP and NP-Complete problems and under Coping with the limitations we come across problems like n-Queens problem, Hamiltonian Circuit problem and Subset Sum problem and under Branch and Bound technique we come across problems like Travelling Salesman,Knapsack,Assignment Problems
